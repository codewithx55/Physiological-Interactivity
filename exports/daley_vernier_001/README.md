# Kenny Session Bundle: daley_vernier_001

Start with `daley_vernier_001.kenny_replay.html`, then open `daley_vernier_001.merged_timeline.csv`. The replay shows whether Vernier force samples overlap the transcript events; the CSV puts session markers, transcript events, audio path, and Vernier force samples on one wall-clock timeline.

- `daley_vernier_001.merged_timeline.csv`: combined timeline for review.
- `daley_vernier_001.breath_word_pairing.csv`: nearest breath sample for each transcript event.
- `daley_vernier_001.kenny_replay.html`: local visual replay of force timing against transcript timing.
- `daley_vernier_001.transcript.csv`: spreadsheet-friendly transcript, when speech events exist.
- `daley_vernier_001.audio.wav`: local audio recording, when enabled.
- `daley_vernier_001.vernier_force.csv`: Vernier force in Newtons with breath phase.
- `daley_vernier_001.markers.jsonl`: session, Vernier start, and Vernier stop markers.
